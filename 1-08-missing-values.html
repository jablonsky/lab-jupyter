How Condition of dropna

How : describes the condition how the NaNs must be dropped.

All = drop the rows where both columns match the conditions of NaN.

Any = drop the rows if any selected column is NaN.

ANY is the default value.

Attribute with the condition for dropping process: any , all ...

In[]:
In[17]:
# all - harder condition to meet
data.dropna(subset=['Zip Code', 'City'], how='all').shape
Out[17]:
(1263, 18)
In[18]:
# easier condition to meet
data.dropna(subset=['Zip Code', 'City'], how='any').shape
Out[18]:
(1259, 18)

First drop all rows that match Zip Code and City condition (NaN) , secondly remove rows that much any condition to Name, Proce and Rating.

Removing data by conditions, is more accurate than dropping everything with NaNs values.

In[19]:
data.dropna(subset=['Zip Code', 'City'], how='all').dropna(subset=['Name', 'Price Range', 'Rating'], how='any').shape
Out[19]:
(1059, 18)

where ( )

Where function can be used to agregate the rows (take row values and if any is NaN , fill the average from columns for this specific row):

notna() - matrix with True where the value exist and False when does not

mean(axis=1) - average aggregated form row

Do you want to go by rows or by columns?

axis=0 - go by columns per each row from Col1 to Coln

in short: if logical test (part 1 ) validate to False, put the value (part 2) , in the order of (part 3).

In[33]:
# where( detect all non NAN value, replace by this value , )
df.where(df.notna() ,df.mean(axis=1) , axis=0)
